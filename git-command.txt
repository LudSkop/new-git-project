Як зрозуміти чи встановився гіт:
git --version ця команда показує версію скачаного гіта 
git --help    ця команда показує які існують команди в гіту
git init      ця команда ініціалізірує репозиторій
git status    ця команда показує текучий статус
git add .     ця команда добавляє всі нові файли і також змінені файли і можна добавити ще окремо файл :git add index.py
git rm --cached index.py  ця команда удаляє файл index.py                                               -----------------
git commit -m "first commit"  ця команда зберігає файл комітить
.gitignore це створили файл в який записуемо все що потрібно заігнорити(всередину файла записуемо)
git branch  ця команда дозволяє зрозуміти в які веткі знаходимось
git branch test ця команда дозволяє создати нову ветку test
git branch -D test ця команда дозволяє удалити ветку test
git checkout test  ця команда дає можливість звернутись до конкретноЇ ветки
git checkout -b new ця команда дає можливість створити нову ветку new і зразу до неЇ звернутись
git merge readme ця команда дає можливість совмістити ветки в даному випадку ми в до ветки  master добавили ветку readme
